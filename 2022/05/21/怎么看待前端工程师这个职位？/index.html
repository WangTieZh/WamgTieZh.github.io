怎么看待前端工程师这个职位？

前端工程师这个职位怎么理解的，它的前景如何？

前端就是网站前台部分，运行在pc端，移动端，等浏览器上展现给用户浏览的东西，从广义上来所有用户在终端产品与视觉和交互有关的部分，都属于前端工程师的专业领域。

狭义上来讲，web前端就是使用html、css、javaScript等专业技能和工具将产品的ui设计稿实现成网站产品，涵盖用户pc端，移动段等网页，处理视觉和交互问题。

前端三剑客，html、css、js

  • html是超文本标记语言，这是我们页面最常用的普通语言，经历了多个版本的发展，已经发展到了5.0版本，得力于W3C建立的标准和规范，普遍升级到了XHTML，XHTML指可拓展超文本标签语言。

  • css 通常称为风格样式表，它是用来进行网页风格设计的。

  • javaScript是客户端脚本语言，为客户提供了更流畅的浏览效果。

前端应用领域。

1、企业官方网站的开发与维护

​ 基本上是纯静态页面的展示，就算是有交互模块，应该也是比较简单的那种。

2、门户类型网站的开发与维护

这种一般分成很多模块，模块之间如何左到复用性强，对产品的更迭很友好，但是很考验开发人员三剑客语言的设计模式。

3、电商网站

例如京东，淘宝，里面不仅有对用户友好的展现方式，合乎逻辑的交互方式，还会设计到网站，商品的色彩设计，排版设计，交互设计。

当然除了单纯的展现，还需要考虑其他的东西 ，如不同端的字体展现，响应式布局，图片大小，静态资源缓存，cdn部署，ajax动态数据交互，还有很多的涉及到前端安全问题，很锻炼人。

4、后台管理系统

这个一般是给内部人员使用的，不用考虑太多的兼容性，直接要求就是统一使用谷歌或者火狐。

5、h5移动端页面

考研开发人员的css功底，要常常写一些动态效果，动画属性，越生动越吸引用户参与。

6、移动端短视频播放页面

最近的短视频崛起，这样的App页面其实就是web前端页面，这样做的原因就是App过审麻烦，且时间周期长，业务需要快速迭代和修复的情况下，前端责无旁贷，那么这时候就要考验的是对视频的操作能力，对流式播放技术和能力的掌握。

7、微信小程序，微信、支付宝、百度等大型企业，先后推出了小程序，身为前端开发者，这是必须掌握的一门技巧，虽然不是一门新的语言，但是代码组织方式，调试方式，底层原理，也是值得关注和学习的。

前端市场现状，

前端入门学习的门槛比较低，所以很多人认为html、css、javaScript是简单学习就可以掌握的东西，以为在网上看点教程随便看看书，用html css 做个网页布局就可以入门了，但是这样的技术水平，其实不够格，在整个市场来看，前端开发的需求量依旧很大，饱和的只是低端饱和，高端始终缺人。

在未来，专业的前端开发工程师才是企业争夺的香饽饽，而被淘汰的不是前端开发，而是技术落后，技术不精的开发者。
