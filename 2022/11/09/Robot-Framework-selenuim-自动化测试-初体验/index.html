Robot Framework selenuim 自动化测试 初体验

Robot Framework 对Selenium, Appium, 等框架语法进行了封装，要求的代码能力降低了, 使用和复用起来也更方便。一套框架可以通吃 GUI，接口，app等（3,4个小时总结出的经验，非常有可能不对）

环境搭建

  • 下载谷歌浏览器Webdriver 提取码 el24 下载完成后需要注册到系统的变量的Path中

  • 安装Ride

    • 安装python3.7版本 下载地址 注意为3.7版本

    • 下载或Clone RIDE项目

    • 解压之后到根目录下执行

    • 安装完成之后会问是不是要在桌面创建一个快捷方式

    • 如果没有创建可以到 RIDE的根目录下运行

    • 安装完成后运行起不来，参考下这个文章 （中文系统的问题）https://www.pythonheidong.com/blog/article/53154/

    • RIDE不能显示中文 https://blog.csdn.net/qq_39184753/article/details/107661789

  • 安装Seleniumlibrary 外部库

    • 文档 Seleniumlibrary 对selenium进行了语义化的封装

    • ```
      pip install –upgrade robotframework-seleniumlibrary
